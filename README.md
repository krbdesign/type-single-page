# type-single-page
 This was my second time using HTML and CSS in my Web Typography course at UMass Dartmouth. This is my first time using sections such as the navigation, the hero, and the header/footer.

# Process
First, I started with researching and picking the font of my choice, Sour Gummy. I chose this font because I wanted to pick a font with a fun name that I could base the entire theme of the website off of. I specifically picked this font name because I felt like I would have a fun time creating a website based off of a type of candy. After picking the font, I created a mood board and a high fidelity mockup on Figma to get a better idea of how I wanted the website to look. After creating the mockup and mood board, I made my ideas come to life in VS code. I also used SVGs exported from my figma files that I had created myself both for the website and the website's favicon. I chose to use the colors I included because I wanted to base my theme off of gummies, but specifically I wanted it to resemble the colors you would find while eating Trolli's sour gummy worms. I chose the playful and simple style because I wanted it to resemble the intended vibe of the font while not taking too much away from it with an excess amount of details and colors.
